# TRUS — Project Context

## What is TruS?

TruS is a modern web development studio that builds high-end, premium websites using React-based
architecture. Unlike platform-dependent solutions (Webflow, WordPress), TruS delivers websites
where clients own every byte — hosting, infrastructure, and codebase.

## Two Concurrent Goals

This repository serves **two purposes simultaneously**, and both must be true at all times:

1. **TruS's own brand website** — A showcase-quality site that communicates TruS's value,
   attracts clients, and demonstrates premium motion-forward frontend engineering.

2. **A commercial React template product** — Every component is designed to be detached from
   TruS-specific content and resold as a premium template. No hardcoded strings in JSX;
   all copy lives in `src/config/site.config.ts`.

## Key Differentiators vs. Platform Solutions

1. Full client ownership — no platform lock-in whatsoever
2. Real React architecture — not a builder export
3. Advanced motion design as a standard feature, not an add-on
4. Scalable, maintainable codebase with documented architecture
5. Template-ready structure enabling white-label resale

## Business Background

(Extracted from `trus_business_overview_developer_deck.md`)

Many businesses want Webflow-quality experiences but cannot accept the ecosystem dependency.
TruS solves this by combining premium visual engineering with full architectural ownership.
The frontend is intentionally over-engineered relative to a typical agency site to serve as
a living proof-of-concept for TruS's capability level.
